From Web Search Relevance to Vertical Search Relevance
Yi Chang
Yahoo Labs, Sunnyvale, CA 94089

yichang@yahoo-inc.com

2.

Web search relevance is a billion dollar challenge, while
there is a disadvantage of backwardness in web search competition. Vertical search result can be incorporated to enrich web search content, therefore vertical search relevance
is critical to provide diﬀerentiated search results. Machine
learning based ranking algorithms have shown their eﬀectiveness for both web search and vertical search tasks. In
this talk, the speaker will not only introduce state-of-theart ranking algorithms for web search, but also cover the
challenges to improve relevance of various vertical search
engines: local search, shopping search, news search, etc.

Yi Chang is a director of sciences in Yahoo Labs, working on web search, vertical search and mobile search challenges. His research interests include information retrieval,
applied machine learning, natural language processing and
social media mining. Yi Chang has co-authored more than
80 referred journal and conference papers, and he has published the first book in this area of vertical search relevance.
He has served as track chair, area chair, poster chair, and
sponsorship chair for SIGIR, WWW and CIKM in the past
a couple of years.
